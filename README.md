# Тестовое задание №1: валидатор списка email-адресов

---

### Цели

Написать простой скрипт на Python, который:

- принимает список email-адресов;
- проверяет наличие MX-записей домена;
- выводит статус для каждого адреса:

  - «домен валиден»;
  - «домен отсутствует»;
  - «MX-записи отсутствуют или некорректны».

---

### Идея

Проверить каждую почту на корректность, затем домены этих почт:

- либо использовать кэш доменов;
- либо добавлять и обрабатывать новый домен, формируя множество уникальных доменов.

Также придерживаться подхода **defence in depth and hard front** - максимально аккуратная обработка ошибок и проверок.

Отсюда получаем:

- аккуратную проверку email-адресов;
- оптимальное время выполнения при повторных запусках за счёт кэша доменов;
- достаточную масштабируемость.

---

### Пошаговое решение

Описание вспомогательных функций, которые:

1. Проверяют валидность расширения входного файла и формата каждого email.
2. Извлекают домен и проверяют его жизнеспособность (существует, не истёк, есть MX-записи).
3. Загружают и используют кэш доменов, при необходимости обновляя его результатами проверки.
4. Обрабатывают входной файл (чтение, проверка, запись результата и обновлённого кэша).
